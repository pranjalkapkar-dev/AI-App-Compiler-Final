## 📌 Overview

AI App Compiler is a **compiler-inspired AI system** that transforms natural language application requirements into structured, executable application blueprints.

It works like a compiler pipeline:

📝 Natural Language  
→ 🧠 Intent Extraction  
→ 🏗️ System Design  
→ 📊 Schema Generation  
→ ✅ Validation  
→ 🔧 Repair  
→ 🖥️ Runtime Simulation  

---

## ✨ Features

- 🧠 Intent Extraction Engine
- 🏗️ System Design Layer
- 🖥️ UI / API / Database Schema Generation
- ✅ Validation Engine (Pydantic + JSON Schema)
- 🔧 Auto Repair Engine for fixing schema issues
- ⚡ Runtime Simulation of generated apps
- 🚀 FastAPI Backend for APIs
- 🎨 Streamlit Frontend for UI interaction
- 🤖 Optional OpenAI integration for smarter outputs
